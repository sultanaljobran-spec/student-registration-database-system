# SQL Queries Explained

## 1. Student Transcript

Shows a selected student's courses by term, including credit hours, enrollment status, and final grade when available.

## 2. Current Courses by Student

Lists course offerings where the student's enrollment status is Enrolled. This is useful for active semester advising.

## 3. Enrollment Count by Course

Counts enrollments across all offerings of each course. This helps identify high-demand courses.

## 4. Enrollment Count by Department

Aggregates enrollment activity by the department that owns each course.

## 5. Average Grade by Course

Calculates average grade points and numeric scores for graded enrollments in each course.

## 6. Average GPA by Student

Calculates a credit-hour weighted GPA for each student using graded enrollments.

## 7. Instructor Teaching Workload

Counts the number of sections and enrolled students assigned to each instructor.

## 8. Students at Risk

Finds students with a calculated GPA below 2.0 so advisors can identify students who may need support.

## 9. Full Course Schedule

Displays semester, course, instructor, section, room, capacity, day, and time for all course offerings.

## 10. Department Performance Summary

Summarizes each department's student count, course count, offering count, and average grade performance.
